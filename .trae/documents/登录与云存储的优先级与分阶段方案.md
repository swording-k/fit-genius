## 操作步骤
1. 初始化 Git（若未初始化）：`git init`
2. 添加远程：`git remote add origin https://github.com/swording-k/fit-genius.git`
3. 创建并切换分支：`git checkout -b dev/initial-sync`
4. 写入 `.gitignore`（Xcode/Swift 模板）
5. 暂存并提交：`git add .` → `git commit -m "chore: initial stable snapshot"`
6. 推送分支：`git push -u origin dev/initial-sync`
7. 打标签并推送：`git tag v0.1.0-stable` → `git push origin v0.1.0-stable`
